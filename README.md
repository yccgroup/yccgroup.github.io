# Theoretical Molecular Science Group website

Static HTML/CSS/JS site prepared for GitHub Pages.

## Files

- `index.html` — homepage
- `research.html` — research areas
- `publications.html` — selected publications with client-side filter
- `people.html` — group members and openings
- `join.html` — recruiting information
- `contact.html` — contact information
- `assets/styles.css` — styling
- `assets/main.js` — mobile menu, current year, publication filter
- `assets/banner.svg` — editable banner inspired by the previous YCC LAB banner
- `assets/favicon.svg` — site icon

## How to publish on GitHub Pages

1. Create a GitHub repository, for example `tms-group.github.io` or any repository name.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set the source to the `main` branch and the root folder.
5. Wait for GitHub Pages to build, then open the URL shown in the Pages settings.
